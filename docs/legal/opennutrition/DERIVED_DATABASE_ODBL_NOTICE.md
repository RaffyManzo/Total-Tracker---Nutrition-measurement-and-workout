# OpenNutrition derived database notice

The static database in this distribution contains information derived
from OpenNutrition.

Attribution: **OpenNutrition** — https://www.opennutrition.app/

The derived database is made available under the Open Database License
(ODbL) 1.0. Individual database contents remain subject to the database
contents licensing terms distributed with the OpenNutrition source
dataset.

Total Tracker modifies the source data by selecting fields, validating
nutrition values, normalizing searchable names, deduplicating records,
generating aliases and dividing the resulting database into static
downloadable shards.

The public distribution must include this notice, the applicable license
texts supplied by OpenNutrition and public access to the complete derived
database represented by the manifest.
